# Checkpointer（状态检查点持久化器）

## 什么是 Checkpointer？

Checkpointer 是 LangGraph 1.x 中的**状态持久化机制**，负责：
- 💾 **自动保存**：每次状态变更时自动存档
- 🔄 **跨会话记忆**：让 Agent 记住历史对话
- ⏪ **时光倒流**：支持回滚到任意历史快照

## 核心概念

- **thread_id**：会话 ID，唯一标识一个用户的对话线程
- **checkpoint_id**：检查点 ID，锁定某个历史时刻
- **config**：时空坐标 `{configurable: {thread_id: "xxx"}}`，决定进入哪个会话
- **StateSnapshot**：历史快照，包含某个时刻的完整状态和 config

## 核心功能

### 1. 跨会话记忆
用户跨越时间再次上线时，使用相同的 `thread_id`，Agent 能回忆起之前的所有对话。

**参考 Demo**：`demo_13_状态检查点.py`

### 2. 时光倒流（状态回滚）
当用户录入错误数据后，可以：
1. 通过 `app.get_state_history(config)` 获取所有历史快照
2. 找到目标快照（如倒数第二个正确状态）
3. 使用该快照的 `config` 重新调用，实现从那个时间点继续对话

**参考 Demo**：`demo_14_记忆回滚.py`

## 核心 API

```python
app.get_state(config)              # 获取当前状态
app.get_state_history(config)      # 获取历史快照列表
```

## 生产环境

- **开发/演示**：`MemorySaver()`（内存存储，重启后丢失）
- **生产环境**：`PostgresSaver` / `SqliteSaver`（持久化存储）

## 关键要点

1. Checkpointer = 时光机，自动保存每个状态快照
2. thread_id 决定进入哪个会话
3. checkpoint_id 决定回到哪个时间点
4. 使用历史 config 调用 = 时光倒流
