# LangChain

## LCEL（LangChain Expression Language）

是一种 LangChain 中用于组装“链 (Chain) 的表达式语言。

<br />

## **Runnable（可运行组件）**

在 LangChain 的世界里，我们会把日志记录、数据库写入都抽象成一个标准的 **Runnable（可运行组件）**



# RAG

## **Embedding（向量化）** 

1. 文档切割 (Chunking)：LangChain 的 `RecursiveCharacterTextSplitter`（递归字符文本切片器）
   1. 会优先按照“段落（`\n\n`）”、“句子（`。`）”来切
   2. 通常每个小段落限制在 300~500 字，并且段落与段落之间要有 50 字的**重叠区（Overlap）**，确保上下文语义不会在切缝中丢失
2. 语义数字化 (Embedding)
