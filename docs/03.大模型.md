# 大模型

## 基本流程
1. 分词器 (Tokenizer)
2. 大模型 (Transformer Model)
    1. 【大模型第一层：Embedding层】： ==TokenIds -> 向量==
    2. 【大模型中间层：Attention等计算层】：几十层神经网络疯狂进行矩阵乘法，让向量与向量之间发生碰撞，理解语义
    3. 【大模型最后一层：LM Head（语言模型头）】：计算结束后，大模型输出一个最终向量。大模型的最后一层神经网络会把这个向量映射回全词表的得分，并通过计算输出概率分布。==转成TokenIds==



## 代码视角

```python
from transformers import AutoTokenizer, AutoModelForCausalLM

# 1. 加载分词器（这只是个词表工具）
tokenizer = AutoTokenizer.from_pretrained("deepseek-ai/DeepSeek-V3")
# 2. 加载大模型本体（这才是那个几十GB、充满参数的神经网络）
model = AutoModelForCausalLM.from_pretrained("deepseek-ai/DeepSeek-V3")

# ---- 运行流程 ----

# 【分词器完成】文字 -> ID (不是向量！)
input_ids = tokenizer.encode("今天天气真") 

# 【大模型独立完成】ID -> 变成向量 -> 内部计算 -> 算出下一个词的概率 -> 挑出概率最高的ID
output_outputs = model.generate(input_ids) 

# 【分词器完成】ID -> 文字
output_text = tokenizer.decode(output_outputs)
```
